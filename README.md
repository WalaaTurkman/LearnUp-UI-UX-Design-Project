# 🎓 LearnUp – UI/UX Design Project

---

## 📌 Overview
LearnUp is a Learning Management System (LMS) designed to provide students with a clean and intuitive experience for browsing courses, managing enrollments, and improving their learning journey.

The goal of this project is to enhance usability, simplify navigation, and create a structured and consistent learning experience.

---

## 👩‍💻 My Role
UI/UX Designer

---

## 🎯 Design Goals
- Improve course discovery experience  
- Simplify navigation between learning sections  
- Create a clean and minimal UI design  
- Ensure consistency across all screens  
- Design scalable layouts for future development  

---

## 🧩 Design Process

### 1. User Flow
- Defined user journey from browsing → enrolling → learning  
- Structured navigation between course pages  

### 2. Wireframes
- Created low-fidelity wireframes to define layout structure  
- Focused on hierarchy and usability  

### 3. High-Fidelity UI
- Designed final screens using Figma  
- Built consistent design system (colors, typography, spacing, components)

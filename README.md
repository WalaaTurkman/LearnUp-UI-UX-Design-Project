# LearnUp UI/UX Design Project

## Overview
A UI/UX design for a Learning Management System (LMS) focused on providing a clean and intuitive experience for browsing courses, managing enrollments, and improving student learning navigation.

## My Role
UI/UX Designer

## Design Process
- Designed user flows for course discovery and enrollment experience  
- Created low-fidelity wireframes to define layout and structure  
- Developed high-fidelity UI screens using Figma  
- Focused on improving usability and navigation between learning features  

## Key Screens
- Latest Courses page  
- My Enrolled Courses page  
- Course browsing interface  
- User course interaction flows  

## Focus Areas
- Clean and minimal UI design  
- Improved user experience for course discovery and learning journey  
- Consistent layout structure for scalable LMS design  
- Ready-to-implement UI for frontend development  

## Tools
- Figma
